# otakuymusic
类似音乐以专辑为单位的维基百科
# 毕业设计:otakuy.com	宅乐维基
***
类似音乐以专辑为单位的维基百科,用户创建好基础主题信息经管理员审核通过后就由上传者来维护这个主题(可由其他用户提交修改由维护者审核后可以合并(最短编辑距离,记录路径 合并 回溯),生成贡献者名单,信息包括专辑简介,tag,听后感)
社区+推荐系统
## 社区:
> 框架选择
* Spring webflux
* Spring security
* Mongodb
	


分为音乐区与讨论区
